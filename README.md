# ☕ Zehra's Beverage Co. - Dynamic Menu Application

A modern, responsive, and high-performance beverage menu application built with **Vanilla JavaScript** and **Tailwind CSS**. This project demonstrates dynamic DOM manipulation, array filtering techniques, and professional UI/UX principles.

## 🚀 Live Demo
[**View Project Live**](https://zehradalgic.github.io/Coffee-Filter/)  
*(Note: Please ensure GitHub Pages is enabled in your repository settings.)*

---

## 🛠️ Tech Stack & Skills
* **HTML5:** Semantic structure for accessibility.
* **Tailwind CSS:** Utility-first styling, responsive design (Mobile-first), and custom gradients.
* **JavaScript (ES6+):** * Dynamic rendering using `.map()` and `.join()`.
    * Category-based filtering with `.filter()`.
    * Event delegation and DOM event listeners.
* **Version Control:** Git & GitHub (Issue-based development workflow).

---

## ✨ Key Features
- **Dynamic Content:** Products are rendered from a data object, making the menu easy to scale.
- **Categorical Filtering:** Instantly filter beverages as "All", "Hot", or "Cold".
- **Responsive Grid:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Modern UI:** Built with a clean coffee-inspired aesthetic and smooth transitions.
